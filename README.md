# Smart Grow 🌱

Smart Grow is an open-source indoor grow automation system based on Arduino/ESP32.

## Goal
Create an affordable and scalable environmental automation platform for indoor grow tents.

## Initial Hardware
- Arduino UNO/Nano
- DHT22 temperature & humidity sensor
- Relay module
- 4” exhaust fan
- Circulation fans
- Samsung LM301B LED panels

## Current Features
- Temperature monitoring
- Automatic exhaust fan control
- Basic environmental monitoring

## Roadmap
- [ ] Automatic ventilation control
- [ ] Humidity monitoring
- [ ] PWM fan control
- [ ] ESP32 migration
- [ ] WiFi connectivity
- [ ] Web dashboard
- [ ] Mobile alerts
- [ ] Irrigation automation

## Vision
Smart Grow aims to become a modular and accessible open-source automation platform for indoor cultivation.
