// 03 · Proyecto

Roadmap

SmartGrow nació de la necesidad de controlar con precisión el ambiente dentro de una carpa de cultivo indoor. La idea es directa: leer sensores, actuar en consecuencia y registrar todo.

El proyecto combina electrónica embebida, programación de firmware y visualización de datos. Diseñado para crecer por etapas, documentado públicamente y construido desde cero.

Cada componente está elegido por su disponibilidad, costo accesible y comunidad activa. El código vive en GitHub y el progreso se sigue en Instagram como @smartgrow.lab.
